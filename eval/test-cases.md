# Тестовые кейсы для /infostyle

10 текстов разных типов. Для каждого указан контекст и ожидаемый уровень строгости.

---

## Кейс 1: Корпоративное описание (строгая)
**Тип:** описание компании на сайте
**Аудитория:** потенциальные клиенты
**Текст:**
```
Наша компания является лидирующим поставщиком качественных образовательных решений. Команда высококвалифицированных профессионалов с индивидуальным подходом к каждому клиенту обеспечивает высокий уровень сервиса и максимально выгодные условия сотрудничества.
```

---

## Кейс 2: UI кнопка (расслабленная)
**Тип:** кнопка в интерфейсе
**Аудитория:** пользователь сервиса
**Текст:**
```
Нажмите на кнопку для осуществления отправки вашей заявки
```

---

## Кейс 3: Сообщение об ошибке (средняя + эмпатия)
**Тип:** error message
**Аудитория:** пользователь после неудачного действия
**Текст:**
```
Ой! Что-то пошло не так. Произошла непредвиденная ошибка. Попробуйте позже или обратитесь в службу технической поддержки для получения дополнительной информации.
```

---

## Кейс 4: Hero-секция лендинга (средняя)
**Тип:** заголовок + подзаголовок лендинга
**Аудитория:** студенты, холодный трафик
**Текст:**
```
Уникальный AI-сервис нового поколения
Наша инновационная платформа использует передовые технологии искусственного интеллекта для создания качественных презентаций и проектных работ любой сложности.
Попробовать бесплатно
```

---

## Кейс 5: Email-рассылка (средняя-строгая)
**Тип:** тема + тело маркетингового email
**Аудитория:** зарегистрированные пользователи
**Текст:**
```
Тема: Невероятная новость для вас!

Дорогие друзья! Мы рады сообщить вам потрясающую новость! Наша команда профессионалов разработала абсолютно уникальную функцию, которая максимально упростит процесс создания ваших работ. Не упустите шанс воспользоваться данным предложением! Переходите по ссылке прямо сейчас!
```

---

## Кейс 6: Форма обратной связи (средняя)
**Тип:** варианты ответа в форме
**Аудитория:** пользователь, не завершивший покупку
**Текст:**
```
Что не подходит?
- Дорого для меня
- Боюсь, что преподаватель "спалит"
- Не устраивает предпросмотр работы
- Дедлайн ещё далеко, сделаю потом
- Другое — напишите в поле ниже
```

---

## Кейс 7: Пустое состояние (расслабленная)
**Тип:** empty state в приложении
**Аудитория:** новый пользователь
**Текст:**
```
У вас пока нет проектов.
```

---

## Кейс 8: Модальное окно подтверждения (средняя)
**Тип:** confirm dialog
**Аудитория:** пользователь, удаляющий проект
**Текст:**
```
Вы уверены?
Вы действительно хотите удалить этот элемент? Данное действие нельзя будет отменить.
Ок / Отмена
```

---

## Кейс 9: Пуш-уведомление (максимальная краткость)
**Тип:** push notification
**Аудитория:** пользователь, ожидающий результат
**Текст:**
```
Уведомление: процесс генерации вашей работы был успешно завершён. Пожалуйста, перейдите в личный кабинет для ознакомления с результатом.
```
